十分钟搞定Pandas

本文是学习Pandas的简要学习笔记。主要参考资料：http://python.jobbole.com/84416/

本文是对pandas官方网站上《10 Minutes to pandas》的一个简单的翻译。这篇文章是对pandas的一个简单的介绍，详细的介绍请参考：CookBook。也可以查看原文。
习惯上，我们会按下面格式引入所需要的包：


In [1]: import pandas as pd

In [2]: import numpy as np

In [3]: import matplotlib.pyplot as plt

In [128]: df["grade"] = df["raw_grade"].astype("category")
